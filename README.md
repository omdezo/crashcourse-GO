# Learning Go

Just me learning Go from scratch. Each file covers something I went through during the course.

## What's in here

- `FirstCode.go` - very first file, just testing the basics
- `printing.go` - printing and string formatting
- `variables.go` - variables, strings, numbers
- `arrays.go` - arrays and slices
- `standardLibrary.go` - standard library stuff like strings and sort
- `Booleans.go` - booleans and conditions
- `loops.go` - loops
- `functions.go` - functions
- `multiple_return_values.go` - multiple return values (Go lets you return more than one thing from a function)

## Notes

I also have `whatilearned.md` where I write down my thoughts and observations after each topic. Worth checking if you want to see what surprised me or what I found interesting along the way.

## Running any file

```bash
go run <filename>.go
```

Or if you want to use the Makefile:

```bash
make
```
